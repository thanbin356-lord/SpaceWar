
# SpaceWar

SpaceWar is a small personal shoot 'em up game project developed with Unity.
The main goal of this project was to strengthen my programming skills and gain more hands-on experience with Unity game development.

In this game, players control a spaceship and fight against waves of enemies in fast-paced arcade-style gameplay.
The project focuses on core game development systems such as:

* Player movement and shooting mechanics
* Enemy AI and spawning system
* Collision detection
* Score system
* UI implementation
* Game flow management
* Visual effects and audio integration

This project was created as a self-learning experience to improve my understanding of C#, Unity workflow, and overall game architecture.

## Technologies Used

* Unity Engine
* C#
* Visual Studio Code
